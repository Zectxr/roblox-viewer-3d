# Roblox 3D Avatar Viewer

A web-based 3D clothing preview application for Roblox avatars, built with React and Three.js.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-View%20Here-blue)](https://zectxr.github.io/roblox-viewer-3d/)

## How to Use

1. Visit the live version: [Roblox 3D Avatar Viewer](https://zectxr.github.io/roblox-viewer-3d/)
2. Upload your Roblox clothing images (PNG/JPG)
3. Select an avatar model from the dropdown
4. View the 3D preview with your clothing applied
5. Export or save your designs

## Features

- Simple and intuitive interface
- High-quality 3D rendering with Three.js
- Multiple avatar models for different styles
- Client-side processing - no data sent to servers
- Responsive design for desktop and mobile
- OBJ model support for custom avatars

## Technologies

- **React** - UI framework
- **Three.js** - 3D rendering engine
- **OBJ Loader** - 3D model loading
- **HTML5 Canvas** - WebGL rendering
